## 项目概述

AIOS Protocol 是一个**跨平台**的 AI 操作系统控制协议实现，让 AI 通过自然语言控制 Windows、macOS 和 Linux 系统及软件。

**支持平台**:
- ✅ Windows 10/11
- ✅ macOS 13+ (Ventura 及以上)
- ✅ Linux (Ubuntu 22.04+, Fedora 38+)

**跨平台实现三阶段目标**:
1. **阶段一**: 系统全面接管 (Q1-Q2 2026)
2. **阶段二**: 通用应用软件接管 (Q3-Q4 2026)
3. **阶段三**: 专业软件的使用和开发支持 (2027+)

## 三阶段目标概览

### Phase 1: 系统全面接管

| 功能类别 | 能力数量 | 技术方案 |
|---------|---------|---------| 
| 系统设置 | 15+ | Node.js + 平台命令行工具 |
| 文件管理 | 10+ | Node.js fs/path 模块 |
| 进程控制 | 8+ | child_process + ps-list |
| 窗口管理 | 8+ | 平台脚本 + robotjs |
| 网络管理 | 6+ | systeminformation + 平台工具 |
| **云端视觉控制 (渐进式)** | 5+ | **Sprint 1: 分析, Sprint 8: 执行** |
| **三层云端 AI** | - | **gpt-4o-mini + gemini + claude** |

### Phase 2: 通用应用软件接管

| 软件类别 | 应用数量 | 控制方案 |
|---------|---------|---------|
| 音乐软件 | 3 | 平台脚本 + API |
| 浏览器 | 3 | Puppeteer / Playwright / CDP |
| 办公软件 | 6 | 平台脚本 + Office API |
| 通讯工具 | 5 | 平台自动化脚本 |
| 邮件客户端 | 3 | 平台脚本 + IMAP/SMTP |
| **开发工具** | - | **AIOS Inspector** |

### Phase 3: 专业软件控制

| 软件类别 | 应用数量 | 控制方案 |
|---------|---------|---------|
| 开发者工具 | 6 | CLI, SDK, API |
| 设计软件 | 4 | UXP, Plugin API |
| CAD/3D | 6 | Python API |
| 音视频 | 3 | 平台脚本 + Python |

## 技术栈

| 组件 | 技术 |
|------|------|
| 核心运行时 | **Node.js + TypeScript** |
| 进程通信 | **stdio / WebSocket / HTTP** |
| GUI 框架 | **Electron / Web UI** |
| AI 引擎 | **三层云端**: gpt-4o-mini / gemini-2.5-flash / claude-sonnet |
| 通信协议 | JSON-RPC 2.0 |
| 包管理 | npm / pnpm |
