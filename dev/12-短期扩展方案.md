## 三、适配器实现详情

### 3.1 SpeechAdapter (语音适配器)

**文件**: `adapters/speech/SpeechAdapter.ts`  
**依赖**: `say` + Google Speech API  
**功能**: TTS 文字转语音 + STT 语音识别

### 3.2 NotificationAdapter (通知适配器)

**文件**: `adapters/notification/NotificationAdapter.ts`  
**依赖**: `node-notifier`  
**平台支持**: Windows ✅ | macOS ✅ | Linux ✅

### 3.3 TimerAdapter (定时器适配器)

**文件**: `adapters/timer/TimerAdapter.ts`  
**依赖**: `node-schedule`  
**功能**: 创建/取消/列出定时器

### 3.4 CalendarAdapter (日历适配器)

**文件**: `adapters/calendar/CalendarAdapter.ts`  
**依赖**: Google Calendar API (OAuth)  
**功能**: Google Calendar 事件管理（需 Google OAuth 授权）

### 3.5 WeatherAdapter (天气适配器)

**文件**: `adapters/weather/WeatherAdapter.ts`  
**API**: OpenWeatherMap  
**功能**: 当前天气、天气预报

### 3.6 TranslateAdapter (翻译适配器)

**文件**: `adapters/translate/TranslateAdapter.ts`  
**API**: Google Translate  
**功能**: 文本翻译、语言检测（需 API Key）

### 3.7 CalculatorAdapter (计算适配器)

**文件**: `adapters/calculator/CalculatorAdapter.ts`  
**依赖**: `mathjs`  
**功能**: 数学表达式计算、单位转换
