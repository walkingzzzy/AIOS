## 一、阶段目标概述

### 1.1 Phase 2 核心目标

**双重目标：**
1. **应用软件控制** - 实现对常用软件的深度控制
2. **协议互操作** - 集成 MCP 和 A2A 协议

| 软件类别 | 目标软件 | 控制方式 |
|---------|---------|---------|
| 音乐软件 | Spotify | Spotify Web API |
| 通讯工具 | Slack, Discord | 官方 API |
| 邮件客户端 | Gmail, Outlook | Web API |
| 办公软件 | Google Docs, Notion, Microsoft 365 (Word/Excel/PPT) | Web API |
| 浏览器 | Chrome, Firefox | Playwright |

### 1.2 协议互操作目标

| 协议 | 集成方式 | 目标能力 |
|------|---------|---------|
| **MCP** | AIOS 作为 MCP Client | 调用 MCP 服务器生态 |
| **MCP** | AIOS 暴露 MCP Server | 让其他 AI 工具调用 AIOS |
| **A2A** | AIOS 实现 Agent Card | 支持 Agent 间协作 |
