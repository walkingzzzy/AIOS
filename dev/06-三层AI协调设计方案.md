## 一、问题分析

### 1.1 当前实现的问题

当前 AIOS 的三层 AI 架构是"单选模式"：

```
用户输入 → IntentClassifier (关键词匹配) → 选择一个模型 → 执行
```

**问题**：
1. 三个模型各自独立工作，没有协调
2. IntentClassifier 只是简单的关键词匹配
3. 没有任务分解和规划能力

### 1.2 目标设计理念

采用"多模型协调"模式：

- **Fast 层**：意图识别、任务路由、简单任务执行 (<500ms)
- **Vision 层**：屏幕理解、元素定位、UI 分析
- **Smart 层**：任务规划、复杂推理、结果汇总

**核心**：三个模型协调工作，而不是互斥选择。

---

## 二、设计目标

### 2.1 三层职责定义

| 层级 | 模型 | 职责 | 响应时间 |
|------|------|------|---------|
| Fast | gpt-4o-mini | 意图识别、工具调用、简单对话 | <500ms |
| Vision | gemini-2.5-flash | 屏幕分析、元素定位、内容识别 | <3s |
| Smart | claude-sonnet | 任务规划、任务分解、复杂推理 | <5s |

### 2.2 协调模式

**模式 A：简单任务（直接执行）**
```
用户 → Fast → 工具调用 → 结果
```

**模式 B：视觉任务（Vision + Fast）**
```
用户 → Vision(分析) → Fast(执行) → 结果
```

**模式 C：复杂任务（三层协调）**
```
用户 → Smart(规划) → [Vision(感知) + Fast(执行)]×N → Smart(汇总) → 结果
```

## 四、核心组件设计

### 4.1 TaskOrchestrator (任务编排器)

```typescript
import { IntentAnalyzer, TaskAnalysis, TaskType } from './IntentAnalyzer';
import { AIEngine, VisionEngine } from './engines';
import { ToolExecutor, ToolCall } from './ToolExecutor';
import { TaskPlanner, ExecutionPlan, ExecutionStep, StepResult } from './TaskPlanner';

interface TaskContext {
  hasScreenshot?: boolean;
  history?: string[];
}

interface TaskResult {
  success: boolean;
  response: string;
  tier: 'direct' | 'fast' | 'vision' | 'smart';
  executionTime: number;
  model?: string;
}

interface OrchestratorConfig {
  fastEngine: AIEngine;
  visionEngine: VisionEngine;
  smartEngine: AIEngine;
  toolExecutor: ToolExecutor;
  toolRegistry: AdapterRegistry;
  smartApiKey: string;
}

/**
 * 任务编排器 - 三层 AI 协调的核心
 */
export class TaskOrchestrator {
  private intentAnalyzer: IntentAnalyzer;
  private fastEngine: AIEngine;
  private visionEngine: VisionEngine;
  private smartEngine: AIEngine;
  private toolExecutor: ToolExecutor;
  private taskPlanner: TaskPlanner;

  constructor(config: OrchestratorConfig) {
    this.intentAnalyzer = new IntentAnalyzer(config.toolRegistry);
    this.fastEngine = config.fastEngine;
    this.visionEngine = config.visionEngine;
    this.smartEngine = config.smartEngine;
    this.toolExecutor = config.toolExecutor;
    this.taskPlanner = new TaskPlanner(config.smartApiKey);
  }

  /**
   * 处理用户请求
   */
  async process(input: string, context: TaskContext = {}): Promise<TaskResult> {
    const startTime = Date.now();
    
    // 1. 分析任务类型
    const analysis = await this.intentAnalyzer.analyze(input, context);
    
    // 2. 根据类型执行
    let result: TaskResult;
    switch (analysis.taskType) {
      case TaskType.Simple:
        result = await this.executeSimple(input, analysis);
        break;
      case TaskType.Visual:
        result = await this.executeVisual(input, analysis);
        break;
      case TaskType.Complex:
        result = await this.executeComplex(input, analysis);
        break;
      default:
        result = await this.executeSimple(input, analysis);
    }
    
    result.executionTime = Date.now() - startTime;
    return result;
  }

  /**
   * 简单任务：Fast 直接执行
   */
  private async executeSimple(input: string, analysis: TaskAnalysis): Promise<TaskResult> {
    // 如果有直接匹配的工具，直接调用（不经过 AI）
    if (analysis.directToolCall) {
      try {
        const result = await this.toolExecutor.execute(analysis.directToolCall);
        return {
          success: result.success,
          response: result.message || '操作完成',
          tier: 'direct',
          executionTime: 0,
        };
      } catch (error) {
        console.error('[TaskOrchestrator] Direct tool execution failed:', error);
        return {
          success: false,
          response: `操作失败: ${error instanceof Error ? error.message : '未知错误'}`,
          tier: 'direct',
          executionTime: 0,
        };
      }
    }
    
    // 否则让 Fast 层处理
    try {
      const response = await this.fastEngine.chat({
        messages: [{ role: 'user', content: input }],
        tools: this.toolExecutor.getAvailableTools(),
      });
      
      // 如果返回工具调用，执行它
      if (response.toolCalls?.length) {
        const toolResult = await this.toolExecutor.execute(response.toolCalls[0]);
        return {
          success: toolResult.success,
          response: toolResult.message || response.content,
          tier: 'fast',
          executionTime: 0,
          model: this.fastEngine.modelName,
        };
      }
      
      return {
        success: true,
        response: response.content,
        tier: 'fast',
        executionTime: 0,
        model: this.fastEngine.modelName,
      };
    } catch (error) {
      console.error('[TaskOrchestrator] Fast layer execution failed:', error);
      return {
        success: false,
        response: `处理失败: ${error instanceof Error ? error.message : '未知错误'}`,
        tier: 'fast',
        executionTime: 0,
      };
    }
  }

  /**
   * 视觉任务：Vision 分析 → Fast 执行
   */
  private async executeVisual(input: string, analysis: TaskAnalysis): Promise<TaskResult> {
    try {
      // 1. Vision 层分析屏幕
      const visionResult = await this.visionEngine.analyzeScreen(input);
      
      // 2. 如果需要执行操作
      if (visionResult.suggestedAction) {
        const actionResult = await this.toolExecutor.executeAction(visionResult.suggestedAction);
        return {
          success: actionResult.success,
          response: actionResult.message || '操作完成',
          tier: 'vision',
          executionTime: 0,
          model: this.visionEngine.modelName,
        };
      }
      
      // 3. 返回分析结果
      return {
        success: true,
        response: visionResult.description,
        tier: 'vision',
        executionTime: 0,
        model: this.visionEngine.modelName,
      };
    } catch (error) {
      console.error('[TaskOrchestrator] Vision layer execution failed:', error);
      return {
        success: false,
        response: `视觉分析失败: ${error instanceof Error ? error.message : '未知错误'}`,
        tier: 'vision',
        executionTime: 0,
      };
    }
  }

  /**
   * 复杂任务：Smart 规划 → 循环执行 → Smart 汇总
   */
  private async executeComplex(input: string, analysis: TaskAnalysis): Promise<TaskResult> {
    try {
      // 1. Smart 层进行任务规划
      const plan = await this.planTask(input);
      
      // 2. 按步骤执行
      const stepResults: StepResult[] = [];
      for (const step of plan.steps) {
        const result = await this.executeStep(step);
        stepResults.push(result);
        
        // 失败处理
        if (!result.success) {
          const decision = await this.handleFailure(step, result);
          if (decision === 'abort') break;
        }
      }
      
      // 3. Smart 层汇总结果
      const summary = await this.summarizeResults(input, stepResults);
      return {
        success: stepResults.every(r => r.success),
        response: summary,
        tier: 'smart',
        executionTime: 0,
        model: this.smartEngine.modelName,
      };
    } catch (error) {
      console.error('[TaskOrchestrator] Complex task execution failed:', error);
      return {
        success: false,
        response: `任务执行失败: ${error instanceof Error ? error.message : '未知错误'}`,
        tier: 'smart',
        executionTime: 0,
      };
    }
  }

  private async planTask(input: string): Promise<ExecutionPlan> {
    // 调用 TaskPlanner
    return this.taskPlanner.planTask(input, this.toolExecutor.getAvailableTools());
  }

  private async executeStep(step: ExecutionStep): Promise<StepResult> {
    if (step.requiresVision) {
      const visionResult = await this.visionEngine.analyzeScreen(step.description);
      return { stepId: step.id, success: true, output: visionResult };
    }
    
    const toolCall: ToolCall = {
      tool: step.action.split('.')[0],
      action: step.action.split('.')[1],
      params: step.params,
    };
    const result = await this.toolExecutor.execute(toolCall);
    return { stepId: step.id, success: result.success, output: result };
  }

  private async handleFailure(step: ExecutionStep, result: StepResult): Promise<string> {
    return this.taskPlanner.handleFailure(step, result.error);
  }

  private async summarizeResults(input: string, results: StepResult[]): Promise<string> {
    return this.taskPlanner.summarize(input, results);
  }
}
```

### 4.2 IntentAnalyzer (意图分析器)

```typescript
import { AdapterRegistry } from './AdapterRegistry';

export interface ToolCall {
  tool: string;
  action: string;
  params: Record<string, any>;
}

export interface TaskAnalysis {
  taskType: TaskType;
  directToolCall?: ToolCall;
  visionPrompt?: string;
  requiresPlanning: boolean;
  confidence: number;
}

/**
 * 增强的意图分析器
 */
export class IntentAnalyzer {
  private toolRegistry: AdapterRegistry;

  constructor(toolRegistry: AdapterRegistry) {
    this.toolRegistry = toolRegistry;
  }

  /**
   * 分析用户输入
   */
  async analyze(input: string, context: TaskContext): Promise<TaskAnalysis> {
    // 1. 尝试直接匹配工具
    const toolMatch = this.matchDirectTool(input);
    if (toolMatch) {
      return {
        taskType: TaskType.Simple,
        directToolCall: toolMatch,
        requiresPlanning: false,
        confidence: 0.9,
      };
    }

    // 2. 检查是否需要视觉
    if (this.requiresVision(input)) {
      return {
        taskType: TaskType.Visual,
        visionPrompt: this.generateVisionPrompt(input),
        requiresPlanning: false,
        confidence: 0.8,
      };
    }

    // 3. 检查是否是复杂任务
    if (this.isComplexTask(input)) {
      return {
        taskType: TaskType.Complex,
        requiresPlanning: true,
        confidence: 0.7,
      };
    }

    // 4. 默认简单任务
    return {
      taskType: TaskType.Simple,
      requiresPlanning: false,
      confidence: 0.5,
    };
  }

  /**
   * 直接匹配工具
   */
  private matchDirectTool(input: string): ToolCall | null {
    // 工具匹配规则
    const patterns: Array<{
      pattern: RegExp;
      tool: string;
      action: string;
      paramExtractor?: (input: string) => Record<string, any> | null;
    }> = [
      // 音量控制
      { pattern: /(调高|增加|提高).*音量/, tool: 'audio', action: 'increase' },
      { pattern: /(调低|减小|降低).*音量/, tool: 'audio', action: 'decrease' },
      { pattern: /静音/, tool: 'audio', action: 'mute' },
      { pattern: /取消静音/, tool: 'audio', action: 'unmute' },

      // 亮度控制
      { pattern: /(调高|增加|提高).*亮度/, tool: 'brightness', action: 'increase' },
      { pattern: /(调低|减小|降低).*亮度/, tool: 'brightness', action: 'decrease' },

      // 应用控制
      {
        pattern: /打开\s*(\S+)/,
        tool: 'app',
        action: 'launch',
        paramExtractor: (input) => {
          const match = input.match(/打开\s*(\S+)/);
          return match ? { name: match[1] } : null;
        },
      },
      {
        pattern: /关闭\s*(\S+)/,
        tool: 'app',
        action: 'close',
        paramExtractor: (input) => {
          const match = input.match(/关闭\s*(\S+)/);
          return match ? { name: match[1] } : null;
        },
      },

      // WiFi 控制
      { pattern: /(打开|开启).*WiFi/, tool: 'wifi', action: 'enable' },
      { pattern: /(关闭|关掉).*WiFi/, tool: 'wifi', action: 'disable' },

      // 电源控制
      { pattern: /锁屏/, tool: 'power', action: 'lock' },
      { pattern: /休眠/, tool: 'power', action: 'sleep' },
      { pattern: /关机/, tool: 'power', action: 'shutdown' },
      { pattern: /重启/, tool: 'power', action: 'restart' },

      // 截图
      { pattern: /截图|截屏/, tool: 'screenshot', action: 'capture' },
    ];

    for (const { pattern, tool, action, paramExtractor } of patterns) {
      if (pattern.test(input)) {
        return {
          tool,
          action,
          params: paramExtractor?.(input) || {},
        };
      }
    }

    return null;
  }

  /**
   * 判断是否需要视觉
   */
  private requiresVision(input: string): boolean {
    const visionKeywords = [
      '屏幕上', '看看', '显示', '界面',
      '点击', '按钮', '菜单', '窗口',
      '图片', '识别', '什么内容',
    ];
    return visionKeywords.some(keyword => input.includes(keyword));
  }

  /**
   * 判断是否是复杂任务
   */
  private isComplexTask(input: string): boolean {
    const complexPatterns = [
      // 多步骤
      /先.*然后/, /首先.*接着/, /第一步.*第二步/,
      // 跨应用
      /和.*比较/, /在.*和.*/, /从.*到.*/,
      // 需要推理
      /分析/, /总结/, /比较/, /评估/, /规划/,
      // 条件判断
      /如果.*就/, /当.*时/,
    ];
    return complexPatterns.some(pattern => pattern.test(input));
  }

  private generateVisionPrompt(input: string): string {
    return `分析屏幕内容，完成用户请求：${input}`;
  }
}
```

### 4.3 TaskPlanner (任务规划器 - Smart 层)

```typescript
import Anthropic from '@anthropic-ai/sdk';

export interface ExecutionPlan {
  goal: string;
  steps: ExecutionStep[];
}

export interface ExecutionStep {
  id: number;
  description: string;
  action: string;
  params: Record<string, any>;
  requiresVision: boolean;
  dependsOn: number[];
}

export interface StepResult {
  stepId: number;
  success: boolean;
  output?: any;
  error?: Error;
}

export type FailureDecision = 'retry' | 'alternative' | 'skip' | 'abort';

export interface ToolDefinition {
  name: string;
  description: string;
  parameters?: Record<string, any>;
}

/**
 * 任务规划器 - Smart 层的核心能力
 */
export class TaskPlanner {
  private client: Anthropic;

  constructor(apiKey: string) {
    this.client = new Anthropic({ apiKey });
  }

  /**
   * 生成执行计划
   */
  async planTask(input: string, availableTools: ToolDefinition[]): Promise<ExecutionPlan> {
    const systemPrompt = `你是一个任务规划器。根据用户的请求，生成一个详细的执行计划。

## 可用工具
${this.formatTools(availableTools)}

## 输出格式
请以 JSON 格式输出执行计划：
{
  "goal": "任务目标描述",
  "steps": [
    {
      "id": 1,
      "description": "步骤描述",
      "action": "tool.action 或 vision.analyze",
      "params": {},
      "requiresVision": false,
      "dependsOn": []
    }
  ]
}

## 规划原则
1. 将复杂任务分解为简单步骤
2. 每个步骤应该是原子操作
3. 标注步骤之间的依赖关系
4. 需要屏幕理解时设置 requiresVision: true`;

    try {
      const response = await this.client.messages.create({
        model: 'claude-sonnet-4-20250514',
        max_tokens: 2048,
        messages: [
          { role: 'user', content: input }
        ],
        system: systemPrompt,
      });

      const content = response.content[0];
      if (content.type !== 'text') {
        throw new Error('Unexpected response type');
      }

      return this.parseExecutionPlan(content.text);
    } catch (error) {
      console.error('[TaskPlanner] planTask failed:', error);
      // 返回一个默认的单步计划
      return {
        goal: input,
        steps: [{
          id: 1,
          description: input,
          action: 'unknown',
          params: {},
          requiresVision: false,
          dependsOn: [],
        }],
      };
    }
  }

  /**
   * 处理步骤失败
   */
  async handleFailure(step: ExecutionStep, error?: Error): Promise<FailureDecision> {
    const prompt = `执行步骤失败：
- 步骤：${step.description}
- 错误：${error?.message || '未知错误'}

请决定下一步操作：
1. retry - 重试当前步骤
2. alternative - 尝试替代方案
3. skip - 跳过当前步骤
4. abort - 终止任务

请以 JSON 格式回复：{"decision": "...", "reason": "..."}`;

    try {
      const response = await this.client.messages.create({
        model: 'claude-sonnet-4-20250514',
        max_tokens: 256,
        messages: [{ role: 'user', content: prompt }],
      });

      const content = response.content[0];
      if (content.type !== 'text') {
        return 'abort';
      }

      const result = JSON.parse(this.extractJSON(content.text) || '{}');
      const validDecisions: FailureDecision[] = ['retry', 'alternative', 'skip', 'abort'];
      return validDecisions.includes(result.decision) ? result.decision : 'abort';
    } catch (parseError) {
      console.error('[TaskPlanner] handleFailure parse error:', parseError);
      return 'abort';
    }
  }

  /**
   * 汇总执行结果
   */
  async summarize(originalRequest: string, results: StepResult[]): Promise<string> {
    const prompt = `用户请求：${originalRequest}

执行结果：
${this.formatResults(results)}

请生成一个简洁、用户友好的结果摘要。`;

    const response = await this.client.messages.create({
      model: 'claude-sonnet-4-20250514',
      max_tokens: 512,
      messages: [{ role: 'user', content: prompt }],
    });

    const content = response.content[0];
    return content.type === 'text' ? content.text : '任务完成';
  }

  private formatTools(tools: ToolDefinition[]): string {
    return tools.map(t => `- ${t.name}: ${t.description}`).join('\n');
  }

  private formatResults(results: StepResult[]): string {
    return results.map(r => 
      `步骤 ${r.stepId}: ${r.success ? '成功' : '失败'} - ${JSON.stringify(r.output)}`
    ).join('\n');
  }

  private parseExecutionPlan(text: string): ExecutionPlan {
    const json = this.extractJSON(text);
    if (!json) {
      throw new Error('Failed to parse execution plan: no JSON found');
    }
    try {
      const plan = JSON.parse(json);
      // 验证必要字段
      if (!plan.goal || !Array.isArray(plan.steps)) {
        throw new Error('Invalid execution plan structure');
      }
      return plan as ExecutionPlan;
    } catch (error) {
      throw new Error(`Failed to parse execution plan: ${error instanceof Error ? error.message : 'unknown error'}`);
    }
  }

  private extractJSON(text: string): string | null {
    // 1. 尝试 markdown 代码块
    const codeBlockMatch = text.match(/```(?:json)?\s*([\s\S]*?)\s*```/);
    if (codeBlockMatch) {
      const candidate = codeBlockMatch[1].trim();
      try {
        JSON.parse(candidate);
        return candidate;
      } catch {
        // 继续尝试其他方法
      }
    }
    
    // 2. 尝试直接匹配 JSON 对象
    const jsonMatch = text.match(/\{[\s\S]*\}/);
    if (jsonMatch) {
      try {
        JSON.parse(jsonMatch[0]);
        return jsonMatch[0];
      } catch {
        // 继续尝试修复
      }
    }
    
    // 3. 尝试修复常见问题
    if (jsonMatch) {
      let cleaned = jsonMatch[0]
        .replace(/,\s*}/g, '}')  // 移除对象尾随逗号
        .replace(/,\s*]/g, ']')  // 移除数组尾随逗号
        .replace(/'/g, '"');     // 单引号转双引号
      
      try {
        JSON.parse(cleaned);
        return cleaned;
      } catch {
        // 解析失败
      }
    }
    
    return null;
  }
}
```

## 六、数据结构定义

```typescript
/**
 * 任务类型
 */
export enum TaskType {
  Simple = 'simple',
  Visual = 'visual',
  Complex = 'complex',
}

/**
 * 任务分析结果
 */
export interface TaskAnalysis {
  taskType: TaskType;
  directToolCall?: ToolCall;
  visionPrompt?: string;
  requiresPlanning: boolean;
  confidence: number;
}

/**
 * 工具调用
 */
export interface ToolCall {
  tool: string;
  action: string;
  params: Record<string, any>;
}

/**
 * 执行计划
 */
export interface ExecutionPlan {
  goal: string;
  steps: ExecutionStep[];
}

/**
 * 执行步骤
 */
export interface ExecutionStep {
  id: number;
  description: string;
  action: string;
  params: Record<string, any>;
  requiresVision: boolean;
  dependsOn: number[];
}

/**
 * 步骤执行结果
 */
export interface StepResult {
  stepId: number;
  success: boolean;
  output?: any;
  error?: Error;
}

/**
 * 任务执行结果
 */
export interface TaskResult {
  success: boolean;
  response: string;
  tier: 'direct' | 'fast' | 'vision' | 'smart';
  executionTime: number;
  executionPlan?: ExecutionPlan;
  stepResults?: StepResult[];
}

/**
 * 失败处理决策
 */
export type FailureDecision = 'retry' | 'alternative' | 'skip' | 'abort';
```

## 九、总结

本设计方案将 AIOS 的三层 AI 从"单选模式"升级为"协调模式"：

1. **Fast 层**：快速响应，直接工具调用
2. **Vision 层**：屏幕理解，元素定位
3. **Smart 层**：任务规划，复杂推理

三层协调工作，而不是互斥选择。简单任务快速响应，复杂任务深度处理。

### 关键成功因素

1. **Phase 1A 是最大价值点**：直接工具匹配可将响应时间从 1-5s 降到 <100ms
2. **渐进式实现降低风险**：每个阶段都有独立价值，可以验证后再继续
3. **跨平台 npm 包支持**：loudness、brightness、nut.js、playwright 等
