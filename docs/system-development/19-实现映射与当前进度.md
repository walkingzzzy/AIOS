## 1. 目标

本文件把“文档里说要做什么”和“仓库里现在真正有什么”逐项对上。  
它用于解决三个问题：

1. 哪些内容已经从概念进入仓库
2. 哪些只是占位文件，尚无实现
3. 当前项目最真实的成熟度在哪里

## 2. 状态分层

| 层级 | 含义 |
|------|------|
| `Spec Only` | 只有文档，无仓库对应物 |
| `Scaffold` | 有 README / schema / YAML / unit 骨架 |
| `Partial Impl` | 有部分实现，但不能形成闭环 |
| `Runnable` | 有最小可运行产物 |
| `Validated` | 已经过 bring-up / E2E / 恢复验证 |
